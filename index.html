GE 299: Web Mapping

Fall 2020, Worcester State University

Dr Alex Tarr

This page contains links to the first web-mapping samples from the Fall 2020 GE299 web mapping class at Worcester State University. Students in the course were given incomplete code via a git repo and asked to edit html, css, and javascript (leaflet library) to visualize a geojson containing the 2020 election results by precinct in Worcester, Massachusettes.
